# aex-bot

<div align="center">
# AEX BOT WhatsApp
</div>

### Install

```bash
> git clone https://github.com/ibnusyawall/aex-bot.git
> cd aex-bot
> npm i
> npm start
```

``Note : runing at port 3000``

#### Usage
<pre>
After scan barcode whatsapp, you can start message with following command : !menu for showing menu/command
</pre>
